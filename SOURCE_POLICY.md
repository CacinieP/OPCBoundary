# 来源政策 | Source Policy

本文档定义 OPCBoundary 仓库中信息来源的引用和管理规则。
This document defines the rules for sourcing and citing information in the OPCBoundary repository.

## 核心原则 | Core Principles

- 所有信息应可追溯到可验证的来源
  All information should be traceable to a verifiable source
- 来源必须合法可获取且可引用
  Sources must be legally accessible and citable
- 存疑时，宁可不收录
  When in doubt, exclude rather than include

## 可接受来源 | Acceptable Sources

信息可来源于以下渠道：
Information may be sourced from:

- 政府官方出版物和监管备案
  Official government publications and regulatory filings
- 公开庭审记录和法律数据库
  Public court records and legal databases
- 主流媒体的新闻报道
  News reports from established media outlets
- 学术出版物和同行评审研究
  Academic publications and peer-reviewed research
- 公开的企业信息披露
  Publicly available corporate disclosures
- 相关实体的官方声明
  Official statements from relevant entities

## 不可接受来源 | Unacceptable Sources

信息不得来源于以下渠道：
Information must NOT be sourced from:

- 合法性不明的泄露文件
  Leaked documents of unclear legality
- 未经可验证核实的匿名爆料
  Anonymous tips without verifiable corroboration
- 内容受限或付费平台（除非合法获取）
  Platforms where content is restricted or paywalled (unless legally accessible)
- 私密社交媒体账号的帖子
  Social media posts from private accounts
- 授权或合法性存疑的任何来源
  Any source where consent or legality is uncertain

## 引用要求 | Citation Requirements

每条数据记录应尽可能包含以下信息：
Each data entry should include, where available:

- 来源名称或出版物 | Source name or publication
- 发布日期 | Publication date
- URL 或引用编号 | URL or reference identifier
- 来源相关性的简要说明 | Brief description of the source's relevance

示例 | Example：

```
来源 | Source: [出版物名称 | Publication Name]
日期 | Date: YYYY-MM-DD
链接 | URL: https://...
备注 | Note: 相关摘录或背景 | Relevant excerpt or context
```

## 存疑来源处理 | Handling Uncertain Sources

如来源的合法性或可靠性存疑：
If a source's legality or reliability is uncertain:

1. 不收录该信息 | Do NOT include the information
2. 如需要可在私下记录中注明不确定性
   Document the uncertainty in a private note if needed
3. 寻找更可靠的来源获取相同信息
   Seek a more reliable source for the same information
4. 仅在来源问题解决后方可收录
   Only include after the source concern is resolved

## 来源复查 | Source Review

- 来源应定期复查其可用性和准确性
  Sources should be periodically reviewed for continued availability and accuracy
- 失效链接应标注或尽可能替换为存档引用
  Dead links should be annotated or replaced with archived references where possible
- 如发现来源不可靠，相关数据应标记为待审查
  If a source is found to be unreliable, the associated data should be flagged for review

## 政策修订 | Amendments

本政策可能随项目发展而更新。变更将通过提交记录追踪。
This policy may be updated as the project evolves. Changes will be tracked in commit history.
